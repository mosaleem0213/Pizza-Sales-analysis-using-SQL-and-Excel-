# Pizza-Sales-analysis-using-SQL-and-Excel-
his project presents a comprehensive analysis of pizza sales data using SQL and visual dashboards to uncover key business insights such as peak ordering times, top-selling products, and customer preferences. With over 21,000 orders and nearly 50,000 pizzas sold, the data reveals that large-sized pizzas and the classic category dominate sales, especially during weekend evenings and lunch hours. SQL queries were used to calculate metrics like total revenue, average order value, and product performance, while visualizations highlight trends across days, hours, and pizza types. The dashboard serves as a powerful tool for understanding consumer behavior and optimizing menu offerings based on data-driven decisions.
 
   **Dashboard Highlights**

- **Total Revenue:** $817,860.05  
- **Total Orders:** 21,350  
- **Total Pizzas Sold:** 49,574  
- **Average Order Value:** $38.31  
- **Average Pizzas per Order:** 2.32  

  **Key Insights**

- **Peak Ordering Times:** 12 PM–1 PM and 6 PM–8 PM  
- **Best Days for Sales:** Friday and Saturday  
- **Top Pizza Size:** Large (59.8% of sales)  
- **Top Category:** Classic (45.2% of sales)  
- **Best Seller:** The Classic Deluxe Pizza  
- **Worst Seller:** The Brie Carre Pizza
